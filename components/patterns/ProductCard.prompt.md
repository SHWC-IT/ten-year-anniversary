A merch card with square image, gold serif price, and a hover-lift Buy button.

```jsx
<ProductCard name="10 Year Anniversary Tee" price="$30.00"
  image="assets/merch-tee.webp" href="https://shwc-store.square.site/..." />
```

Lay them out in a CSS grid (3-up or 5-up). Set `external={false}` for an in-app Buy handler via `onBuy`.
