The signature double-bordered gold frame; wrap CTA panels, giving cards, and value cards in it.

```jsx
<FramedPanel align="center" glow>
  <Eyebrow align="center">Giving</Eyebrow>
  <SectionTitle align="center" script="decade.">Sow into the next</SectionTitle>
  <Button size="lg">Give Now</Button>
</FramedPanel>
```

`glow` adds a radial gold wash from the top. Compose Eyebrow / SectionTitle / Button inside.
