A gold-on-navy call-to-action button (tracked-caps, sharp corners); use for every primary or secondary action.

```jsx
<Button href="giving.html">Give Now</Button>
<Button variant="ghost" size="sm">Add to Calendar</Button>
<Button size="lg" wide>Register</Button>
```

Variants: `primary` (gold gradient) · `ghost` (hairline outline, gold on hover). Sizes: `sm` · `md` · `lg`. Props: `wide`, `disabled`, `href` (renders an anchor). Pair an inline external-link SVG as a child for off-site links.
