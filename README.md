# SHWC 10 Year Anniversary Site (MVP)

Deploy to Netlify: drag this whole folder into https://app.netlify.com/drop
(or connect a repo and set publish directory to this folder).

Pages:
- index.html  (Home / Anniversary)
- about.html  (Values & Mission)
- giving.html (Giving)
- store.html  (redirects to the external store)

Note: Giving checkout is a front-end mockup; Stripe needs to be wired up before taking real payments. The Store is hosted externally, all "Store" / "Buy Now" / "Shop the Collection" links point to the external store URL (set in the page source, currently a placeholder: https://shwc-store.square.site).
